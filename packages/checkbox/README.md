## Checkbox 多选框

#### 1、Api

| 属性 | 描述 | 类型 | 默认值 | 可选项 | 
|  ----  |  ----  | ----  | ----  | ----  |
| label | 标签 | String | "" | 无
| value | 绑定值 | Boolean、undefined | undefined | 无
| defaultValue | 默认值 | Boolean、undefined | undefined | 无
| disabled | 是否禁用 | Boolean | false | true 、false
| indeterminate | 半选中状态 | Boolean | false | true、false
